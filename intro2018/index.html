    Færdig?

    Opfattet som en funktion er fakultet kun defineret for ikkenegative hele tal
    https://da.wikipedia.org/wiki/Fakultet_(matematik)

    Hvordan klarer vi den?

    
def fakultet(n)
   if(n < 0) then
      return "Ikke defineret"
   end

   sum = 1
   for i in 1..n
      sum *= i
   end
   return sum
end

puts "-10! = #{fakultet(-10)}"
